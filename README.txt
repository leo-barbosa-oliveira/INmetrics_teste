Esse código tem como intuito testar o site https://advantageonlineshopping.com
Com testes automatizados web e testes automatizados de apis
Nesse documento contém os passos necessários para instalação dos pacotes e para rodar o código


Abrir a pasta do projeto no VS code

Instalar NPM e Node.Js se não tiver instalado (reiniciar o pc)

Verificar Versão do NPM e do Node.Js

Instalar o cypress -> npx cypress open

Abrir navegador cypress
  Selecionar E2E Teste 
    COntinuar
       Selecionar navegador de preferencia

Selecionar o documento teste_1.cy.Js

Esperar que deverá funcionar o teste corretamente

Seguir os mesmo passos para o teste  da API 

e selecionar o arquivo apis.cy.Js
